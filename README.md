# hermes-ontology-manager 🕸️
> **No solo guardo datos, gestiono conocimiento estructurado.**

Como estudiante de Inteligencia Artificial en ITLA, entiendo que los LLMs son poderosos, pero el conocimiento estructurado es lo que da precisión. Este manager permite cargar archivos `.ttl` o `.owl`, realizar consultas **SPARQL** y mantener todo respaldado en mi "Second Brain".

## 🚀 Funcionalidades Clave
- **Consultas SPARQL:** Interroga tu base de conocimientos de forma semántica.
- **Notion Sync:** Backup automático de entidades y relaciones a bases de datos de Notion.
- **Obsidian Bridge:** Generación de notas en Markdown a partir de grafos de conocimiento.
- **Soporte RDF/OWL:** Manejo de estándares de la Web Semántica.

## 🛠️ Stack Técnico
- **Python** (rdflib, requests)
- **SPARQL** para queries dinámicas.
- **Notion API** para integración de flujos de trabajo.

## 📈 Valor Profesional
Este proyecto demuestra mi capacidad para diseñar arquitecturas de datos complejas, uniendo la flexibilidad de la IA con la rigidez necesaria de las bases de datos semánticas. Ideal para sistemas de RAG (Retrieval-Augmented Generation) avanzados.
